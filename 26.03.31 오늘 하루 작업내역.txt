  4. 통역사 단가 관리 시스템 (Feature 1~4)
═══════════════════════════════════════════════════════════════

[테스트 4-1] Feature 1: 단가 범위 제한
  - 관리자 → 시스템 설정 → "통역사 단가 범위 설정" 카드
  - 기대: 서비스 유형별(부스/미팅/컨퍼런스/현장운영) 최소/최대 단가 설정 가능
  - 통역사 대시보드: 각 단가 입력 아래 "허용 범위: OOO원 ~ OOO원" 표시
  - 범위 밖 금액 입력 후 저장 시 에러 토스트로 차단
  - DB: 90_시스템설정 key='rate_limits'

[테스트 4-2] Feature 2: 단가 변경 알림 + 이력
  - 통역사가 단가 변경 후 저장 시 91_단가변경이력 테이블에 이전/이후 값 기록
  - 관리자에게 24_알림 테이블로 알림 전송
  - 관리자 → 통역사 상세 모달 → "📊 단가 변경 이력" 섹션에 변경 기록 표시
  - 변경된 값은 빨강(이전)→초록(이후) 하이라이트

[테스트 4-3] Feature 3: 단가 승인 워크플로
  - 통역사 대시보드: "프로필 정보 저장" / "단가 변경 신청" 버튼 분리
  - 단가 변경 신청 → pending_rate_by_type/pending_rate_by_language에 저장
  - rate_status = 'pending' → 노란 배너 "승인 대기 중" + 입력 비활성화
  - rate_status = 'rejected' → 빨간 배너 + 반려 사유 표시
  - 관리자: "단가 승인대기" KPI 카드 + 필터 칩
  - 관리자: 통역사 상세 → 현재 vs 요청 단가 비교 테이블
  - ✅ 승인: pending → rate_by_type 복사, 이력 기록, 통역사에 알림
  - ❌ 반려: 사유 입력 후 반려, 통역사에 알림
  - DB: 40_통역사프로필에 pending_rate_by_type, rate_status 등 컬럼 추가

[테스트 4-4] Feature 4: 경력별 추천 단가
  - 관리자 → 시스템 설정 → "경력별 추천 단가 설정" 카드
  - 3단계: 주니어(1~3년) / 경력(3~7년) / 시니어(7년 이상)
  - 각 단계별 4가지 서비스 유형 최소/최대 추천 단가 설정
  - 통역사 대시보드: 본인 경력에 맞는 추천 범위 표시
    - ✅ 초록: 추천 범위 내
    - 🟡 노랑: 약간 초과 (10만원 이내)
    - 🔴 빨강: 크게 초과 (10만원 이상)
  - 단가 입력 변경 시 실시간으로 색상 업데이트
  - DB: 90_시스템설정 key='recommended_rates'
