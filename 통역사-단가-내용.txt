[1] 데모 통역사 단가 시장가 조정
--------------------------------------
- 수정 파일: index.html, interpreters.html
- 기존 단가: 부스 기준 30~45만원 (시장 대비 과다 설정)
- 변경 기준: 전시회 통역 시장 실거래가

  경력별 단가 (부스 상주 기준, 일반 언어):
  - 신입/주니어 (1~3년): 20~25만원/일
  - 경력 (3~7년): 25~30만원/일
  - 시니어 (7년+): 30~35만원/일

  희소 언어 프리미엄 (독일어, 아랍어):
  - 각 구간 +5만원 추가

  통역 형태별 단가 차이:
  - 부스 상주 < 미팅 동행 < 컨퍼런스 < 현장 운영
  - 각 형태별 약 5만원 차이

  실제 등록 통역사 (김하나, 경력 3년):
  - 부스 25만원 / 미팅 30만원 / 컨퍼런스 40만원 → 변경 없음


[2] 경력별 단가 자동 추천 시스템
--------------------------------------
- 수정 파일: interpreter-dashboard.html
- 통역사 대시보드 프로필에 "통역 경력 (년)" 입력 필드 추가
- "경력별 단가 추천" 버튼 클릭 시 자동 단가 설정:

  1~3년 (주니어):
    부스 200,000 / 미팅 250,000 / 컨퍼런스 300,000 / 운영 220,000

  3~7년 (경력):
    부스 250,000 / 미팅 300,000 / 컨퍼런스 350,000 / 운영 270,000

  7년+ (시니어):
    부스 300,000 / 미팅 350,000 / 컨퍼런스 400,000 / 운영 320,000

- 추천 단가는 통역사가 직접 수정 가능
- 프로필 저장 시 DB(40_통역사프로필)에 자동 연동
  → 경력, 이름, 연락처, 자기소개, 언어, 전문분야, 단가 모두 DB 반영


[3] 경력 사칭 방지 — 관리자 인증 시스템
--------------------------------------
- 수정 파일: interpreter-dashboard.html, admin-dashboard.html
- Supabase: interpreter-docs Storage 버킷 생성, 40_통역사프로필 필드 추가

  (A) 통역사 측 (통역사 대시보드 > 프로필)
  - 경력 인증 상태 표시 (대기/완료)
  - 증빙 서류 업로드 기능 (경력증명서, 자격증, 포트폴리오)
    - 파일 형식: PDF, JPG, PNG
    - 최대 크기: 10MB
  - 업로드된 서류 목록 표시 + 삭제 가능

  (B) 관리자 측 (관리자 대시보드 > 통역사 관리 탭)
  - 등록된 통역사 목록 (경력, 단가, 언어, 인증 상태)
  - 인증 상태 배지:
    - "미제출" (회색) — 서류 미제출
    - "서류검토 필요" (주황) — 서류 제출됨, 검토 대기
    - "인증완료" (녹색) — 관리자 승인 완료
  - 관리자 기능:
    - 제출된 서류 열람 (새 탭에서 열기)
    - "인증 승인" 버튼 → is_verified = true, verified_at 기록
    - "반려" 버튼 → 사유 입력 후 반려 처리
    - "인증 해제" 버튼 → 기존 인증 취소 가능

  (C) 플로우
  1. 통역사 가입 → 경력 입력 + 증빙 서류 업로드
  2. 관리자가 서류 검토 → 인증 승인 또는 반려
  3. 인증된 통역사 → 신뢰할 수 있는 단가로 확인
  4. 인증 안 된 통역사 → "인증 대기" 상태 표시


[4] 플랫폼 수수료 구조
--------------------------------------
- 통역사 수령액: 단가 100% (수수료 없음)
- 고객 결제액: 단가 + 플랫폼 수수료 10%
- 콘텐츄어 수익: 플랫폼 수수료 10%

  예시) 통역사 부스 단가 250,000원/일
    - 통역사 수령: 250,000원
    - 고객 결제: 275,000원 (250,000 + 수수료 25,000)
    - 콘텐츄어: 25,000원

- 통역사에게 수수료를 부과하지 않는 이유:
  → 초기 단계에서 통역사 확보가 우선
  → 이중 수수료(양쪽 부과)는 통역사 이탈률 증가
  → 플랫폼 성장 후 통역사 수수료(3~5%) 도입 검토 가능


[5] 향후 계획
--------------------------------------
- 플랫폼 실적 기반 자동 등급 시스템 (통역사가 많아진 후 도입)
  → 콘텐츄어 내 완료 건수 + 고객 평점으로 등급 결정
  → 0~5건: 신입 단가, 5~15건: 경력 단가, 15건+: 시니어 단가
  → DB에서 자동 계산되므로 경력 조작 불가
