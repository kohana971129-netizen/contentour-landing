■ 1. 관리자 단가 관리 기능 강화
─────────────────────────────────
[파일] admin-dashboard.html

(1) 관리자 직접 단가 조정 기능
  - 통역사 상세 모달에서 "✏️ 관리자 직접 단가 조정" 버튼 추가
  - 유형별(부스/미팅/컨퍼런스/현장) + 언어별 가산 단가 입력 폼
  - 조정 사유 필수 입력 → 승인 대기 없이 즉시 반영
  - 통역사에게 단가 조정 알림 자동 발송

(2) 사이드바 승인대기 알림 뱃지
  - "💰 통역사 단가 관리" 메뉴 추가 (→ interpreters 탭 연결)
  - 승인대기 건이 있으면 주황색 뱃지(N건) 표시

(3) 승인/반려 감사 로그 보강
  - DB 91_단가변경이력 테이블에 4개 컬럼 추가:
    · action_type (approved / rejected / admin_edit)
    · actor_id (처리 관리자 ID)
    · actor_name (처리 관리자 이름)
    · reason (사유)
  - 승인/반려/관리자 조정 시 감사 로그 자동 기록
  - 이력 테이블에 구분/처리자 컬럼 추가 표시

  - 상담일지 검토 완료 시 → "📝 상담일지 검토 완료" 알림
